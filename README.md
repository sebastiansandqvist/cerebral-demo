# cerebral-demo

## Installation:

1. **Clone the repo**

SSH:
```bash
git clone git@github.com:sebastiansandqvist/cerebral-demo.git
```

or

HTTPS:
```bash
git clone https://github.com/sebastiansandqvist/cerebral-demo.git
```

2. **Install dependencies**

```bash
npm install
```

3. **Start the dev environment**

```bash
npm run part1
```

Later on...

```bash
npm run part2
```

```bash
npm run part3
```
